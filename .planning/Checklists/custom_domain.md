## GitHub Pages Custom Domain

- [ ] Add custom domain in repository settings. Create a TXT record.
- [ ] Commit `CNAME` (if required).
- [ ] Configure DNS A/CNAME records.
- [ ] Wait for propagation.
- [ ] Verify HTTPS certificate.
- [ ] Test:
    - http://
    - https://
    - www.
    - apex domain
- [ ] Get the base URL setup in TOML
